Управление рестораном

Приложение состоит из модулей:
1) Управление столиками (CRUD). Добавить фильтр по имени

2) Управление официантами (CRUD). Добавить фильтр по имени

3) Управление блюдами меню (CRUD). Добавить фильтр по категории

4) Модуль счетов. Официант может «открыть» счет для определенного столика и официанта, добавлять в него какие-то блюда и затем «закрыть». Нужно посчитать общую сумму счета

Сервис для генерации api https://www.mockapi.io/
